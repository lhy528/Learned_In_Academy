CSS Position 속성 5가지 값 : 위치를 지정하는 속(left, right, top, bottom 속성 사용 가능) static : 태그에 위치값을 지정할 수 없음. 방위값을 넣어도 반영 안됨(문서에 고정) absolute : 문서의 좌측 상단을 기준으로 절대위치를 나타냄 relative : static인 태그를 기준으로 위치 지정 fixed : 문서의 좌측 상단을 기준으로 요소를 고정. 스크롤과 상관 없음(화면에 고정) inherit : 부모 태그의 속성값을 종속
