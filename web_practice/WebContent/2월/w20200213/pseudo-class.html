기본 선택자 외의 요소에 CSS효과를 주고 싶을 때
특정한 상태 말고 규칙에 따라 상태가 바뀌는 요소를 선택할 때

선택자:pseudo-class(속성:값);

많이 사용되는 요소

요소 설명
:link 링크 변경
:visited 한번 이상 방문한 링크 변경
:hover 마우스 커서가 링크 위에 올라갔을 때 변경
:active 활성화된 요소에 대한 속성 지정
:focus 포커스를 받은 요소에 대한 속성 지정
:first-child 첫재자식에대한 ...
:lang 특정 language에 대한 ...
:not 지정한 항목을 제외한 모든 항목을 선택
:target 앵커를 이용해서 웹 페이지 안에서 타깃이 되는 부분 표시
